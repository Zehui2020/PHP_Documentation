Company Stuff

  1. Seafood UAT

  2. Fresh UAT

  3. UAT: User acceptance testing: Testing system among users.

  4. SIT: System Integration Testing: Testing system among devs.

  5. PMT: Prime Mart Trading.

  6. MSOH: Min stock at hand.

  7. PLU: Price Lookup (barcode, etc.)

  8. Eleap: Weighing system, weigh food, get barcode.

  9. IBT: Price Prime sells to supermarkets.

  10. RSP: Recommended Selling Price.

  11. UOM: Unit Of Measurement

  12. Sage: Database for fresh goods like fruits & vegs

  13. Navision: Database for other goods (like pencils, etc.)

  14. MAHOTA Food, Prime Online, Prime Prime: Not related to Prime. Mahota, Prime Online is e-commerce.

  15. Sync: Since the 3 companys above are ordering from Prime as well, need to sync with other companies. Prime Online is for corperate customers, Prime Prime for normal consumers.

  16. MOQ: Minimum Order Quantity.

  17. PO / PU: PO is list of things buyers bought from outside vendors / PU is list of things prime sell to supermarket.

  18. Special Pricing: Top priority for pricing. For giving discounts to certain customers.

  19. DO: Delivery Order generated when customer recieves the product.

  20. 12:00 - 14:00 & 16:00 - 17:00: Same day delivery.

  21. 00:00 - 10:00 & 12:00 - 14:00 & 16:00 - 17:00: Next day delivery.

  22. Add on & Declaring stock balance & Force allocation: Ordering for seafood.

  23. Normal Ordering & Standing Order & Allocation Order: Ordering for fresh.

    1. Normal Order is similar to Add On, but diff cuz outlet has to select individually and "add to cart".
    2. Buyer can allocate to Outlet. Outlet can reject allocation order.
    3. Standing Order can be set based on customer behaviour, e.g. every monday, etc.

  24. Block Normal Order: If don't have stock, can block the order so customer don't order the product.

  25. Cutoff Time: Did by intern. Can set num of days to block order. During block days, cannot order. Used for standing order.

  26. Must set delivery time for customer in pmt fresh when creating it.

  27. Account: Knows how much PMT buys from Supermarket (IBT)

  28. PSM Account: Can't know how much PMT buys from Supermarket (IBT)

  29. Con job: Setup certain time, certain code to be ran.

  30. CMS: Content Managment System.

  31. SSL: Secure Sockets Layer. Encrypts data btw server & website.

  32. http: port num 80 defualt. Not secure, outsider can listen in.

  33. https: Port num 443 default. More secure, outsider cannot listen in.

  34. middleman attack: Same website, diff URL.

  35. Intranet: Any new from HR post onto notice board. Using Jun Lao.

  36. DNS: Domain Name System.

  37. IP Address: Internet Protocol. A unique number that gets linked to all online activity you do

  38. IPv4: Uses 32-bit address: 2^32 of possible adresses.

  39. IPv6: Uses 128-bit address: 2^64 possible adresses.

  40. Prime Fish Farms: 3 total: Pasir Ris, Kelong, Katam

  41. WMS: Warehouse Managment System.

  42. IPS Project: WMS/PGI. Combine every ordering into 1 system.

  43. JLPI: Old packeting system.

  44. SOP: Standard Operating Procedures.

  45. Prime Rewards QR: 3 Types: Member, Prime, Coupon QR.

  46. Prime Points: 200 pts -> $1. $20 -> 1 stamp. Can only get 1 daily. Resets monthly. If have max scratches and you earn more, the extras will be ignored.

  47. POS: Point Of Sale : a place where something is sold to the public, or the place where someone pays for something.
